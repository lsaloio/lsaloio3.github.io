Checking for color contrast

The Web Content Accessibility Guidelines require websites to have a color contrast ratio of at least 4.5:1 to maintain readability. There are multiple resources available to confirm your color palette adheres to WCAG standards.

Finding accessible color combinations with Color Safe

  1. Go to http://colorsafe.co/ and click Get Started.

  2. Click the Background Color textbox and enter the hex code for your desired background color.

  3. Click Generate Color Palette. A menu appears with suggested colors for text.

  4. Select a color from the menu, the bar at the top shows an example of text with the selected background color and text color. Next to the sample display is the contrast ratio. Please note Color Safe only provides the contrast ratio, it is up to users to confirm the color combination adheres to WCAG either from knowledge or validating with a second source such as WebAIM or WAVE.

Checking with WebAIMâ€™s contrast checker

  1. Go to https://webaim.org/resources/contrastchecker/

  2. Click the text box in the Foreground section. Type the hex code for the desired text color for your site here.

  3. Click the text box in the Background section. Type the hex code for the desired background color of your site here.

  4. Click outside both sections, the Current Ratio box updates with the contrast ratio of the two colors selected. A green border appears with a successful contrast ratio.

  5. To explore other ratios, toggle with the Lightness value in either section to experiment with different values of each selected color. If your ratio falls just under the guideline this is a way to find the most similar color combination that remains accessible.

Checking contrast on a published site with WAVE

  1. Copy and paste the address for the website you want to check.

  2. Go to https://wave.webaim.org/.

  3. In the Web page Address box paste the desired web address and click enter. The page loads with an analysis of the website entered.

  4. In the side menu, the section Contrast Errors notes if any elements on the web page are non-compliant with WCAG. Zero errors is a successful and compliant contrast ratio. If you have an error, click the Details menu in the side menu to see what specific elements are causing poor contrast and need to be adjusted.
